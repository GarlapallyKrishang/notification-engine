# Spring Boot Notification Engine - Frontend

This is the Next.js React frontend for the Notification Prioritization Engine evaluating MERN stack vs Spring Boot stack.

*(Note: This codebase is structurally identical to the MERN frontend as required by the spec to demonstrate seamless API contract parity).*

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)
